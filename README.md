# Financial-Statistics-Dashboard-Excel
#  Interactive Income & Marketing Performance Dashboard (Excel)

This project features a fully dynamic Excel dashboard designed to analyze income sources, profitability, and marketing strategy performance.  
Every chart and KPI updates instantly based on the selected year, creating a seamless interactive analysis experience.

The dashboard breaks down all income streams, shows contribution percentages, compares results against targets, and visualizes monthly financial performance — turning raw financial data into clear, meaningful business insights.

---

##  Project Overview

This dashboard includes several interactive and analytical components:

- **Dynamic Web Chart** analyzing all income sources and their percentage contribution  
- **Automatic Highlighting** for the highest income source  
- **Creative Donut Chart** showing the achieved percentage of the annual target  
- **Total Income & Target KPIs** for quick performance evaluation  
- **Month-wise Income Trend Chart** with clean layout and alignment  
- **Income Source Summary** displaying counts and percentage split  
- **Average Monthly Income Metric**  
- **Monthly Operating Profit Bar Chart** with total amount visualization  
- **Marketing Strategy Analysis** comparing two types of marketing approaches  
- **Year-Based Slicer** that dynamically updates all visuals, metrics, and calculations  

This dashboard provides a 360° view of income, target achievement, profitability, and marketing effectiveness.
